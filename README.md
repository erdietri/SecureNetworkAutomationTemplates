# SecureNetworkAutomationTemplates
There's plenty of repeatablity in network automation; so why should we all write the same scripts separately, from scratch? This repository contains a library of single-file network automation templates that follow secure code practices. Choose one as a skeleton for your next network automation project, or use them as context when working with an AI agent.

## Requirements
* VS Code 1.99.3 (or IDE of your choice)
* Python 3.13.3 (another Python3 version may also work, but only 3.13.3 has been tested at this time)
* Python libraries listed in requirements.txt (see Run Scripts for additional info)

## Installation
* If you do not have VS Code or another IDE, you may download and install VS Code for your OS here: https://code.visualstudio.com/download.
* If you do not have Python3, choose your OS and download python here: https://www.python.org/downloads/. Then, follow the installation instructions here based on your OS: https://kinsta.com/knowledgebase/install-python/#how-to-install-python.

## Run Scripts
* Clone this repository: git clone https://github.com/erdietri/UmbrellaUncommonDomains.git
* Navigate to the project directory: cd securenetworkautomationtemplates
* Further navigate to the folder containing the script you're interested in (important, as each template is in its own directory with its own requirements file)
* Install the dependencies: pip install -r requirements.txt
* Add any variables required in the .env file for the script. (Simply copy/paste. They do not need to be formatted as strings.)
* Run the Python script: python3 INSERT_NAME_OF_SCRIPT.py
